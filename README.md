# Rock-Paper-Scissors Game (Python)

A simple and interactive **Rock–Paper–Scissors** console game developed using Python.  
This project demonstrates core programming concepts in a clean and beginner-friendly way.


# Features

- User input for rock, paper, or scissors  
- Random computer selection  
- Game logic to determine win, loss, or tie  
- Displays round results clearly  
- Score tracking across multiple rounds  
- Option to replay the game  


# Technology Used

- **Programming Language:** Python  
- **Library:** `random`


# How to Run

1. Install Python (version 3.x recommended)  
2. Clone or download this repository  
3. Open a terminal in the project folder  
4. Run the program:

```bash
python rock_paper_scissors.py
